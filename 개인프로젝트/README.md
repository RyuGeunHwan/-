# 호텔 예약 및 예약 확인

## backlog

```
1. 고객은 입력란을 모두 작성하여야 한다.
2. 고객은 날짜를 선택하여야 한다.(단, 과거일 경우 X)
3. 고객은 인원수를 체크 하여야 한다.(인원수 초과시 금액 추가)
4. 고객은 예약을 하게되면 예약번호로 예약 정보를 확인 할 수 있어야 한다.
5. 관리자는 고객 예약시 예약 확인 날짜를 세부적인 내용을 문자로 자동전송 해주어야 한다.
```

## 기능

```
1. 예약페이지 이동
2. 방정보(사진,가격,객실 정보,편의 시설,주변 시설,세부설명)
3. 입력란
    3-1 원하는 방선택(방정보 슬라이드로 보여주기),성명,주민등록번호,핸드폰번호,체크인,체크아웃,인원수선택(어린이,성인)
```
